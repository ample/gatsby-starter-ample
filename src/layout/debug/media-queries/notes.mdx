# Debug Media Queries

This component provides a quick way to find the current media query for faster local development.

## Basic usage

Download [direnv](https://direnv.net) via [homebrew](https://brew.sh)

```
  brew install direnv
```

Create a `.envrc` file in the root directory and add the following

```
  export GATSBY_DEBUG_MEDIA_QUERIES=true
```

Run `direnv allow` and restart the development server.

_The media query degugger is located in the bottom left of the browser durning local development_
