# Button

A button is a styled link. It accepts a `url` and `label` and renders a clickable object on the screen. It will also accept a `className` property, which can be used to customize styling.

The button component sits on top of the [Component Library's `<Link />` component](/?path=/story/components-link), which adds logic to determine whether we can take advantage of Gatsby Link or not.
